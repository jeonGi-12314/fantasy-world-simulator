# ⚔ Fantasy World Simulator

> 캐릭터들이 살아 숨쉬는 판타지 세계를 운영하는 텍스트 기반 시뮬레이터

**[▶ 플레이하기](https://fws-simulator.surge.sh)**

---

## 개요

Fantasy World Simulator는 플레이어가 직접 캐릭터를 설계하고, 그들이 스스로 모험을 떠나며 성장하는 과정을 지켜보는 **방치형 시뮬레이션 게임**입니다.  
캐릭터들은 매일 서로 관계를 맺고, 직업을 바꾸고, 연애를 하고, 전쟁에 참여하며 세계의 운명을 결정짓습니다.

---

## 주요 기능

### 🧑 캐릭터 시스템
- **6가지 스탯** — 전투(STR), 마법(INT), 신성(FAI), 민첩(AGI), 사교(CHA), 생존(END)
- **20포인트 자유 배분** 또는 🎲 랜덤 롤
- **성별 / MBTI / 성향(Light·Neutral·Dark)** 설정
- **이모지 아이콘 선택** — 남성·여성·기타별로 다른 아이콘 풀
- **정신 상태** (안정 / 불안 / 외상 / 결의) 초기 설정

### ⚔ 12가지 직업 (전직 시스템)
스탯과 행동 횟수 조건을 충족하면 자동 전직 제안이 발생합니다.

| 직업 | 조건 | 역할 |
|------|------|------|
| 전사 ⚔ | STR 6, 전투 5회 | 토벌 의뢰, 경비 |
| 마법사 🔮 | INT 6, 마법 4회 | 마법 재료 수요 생성 |
| 성직자 ✝ | FAI 6, 신앙 4회 | 치유 서비스 |
| 도적 🗡 | AGI 6, 은신 4회 | 암시장 접근 |
| 기사 🛡 | STR·FAI 5, 전투 3회 | 귀족 의뢰 |
| 음유시인 🎵 | CHA 6, 사교 5회 | 공연 수입, 시장 정보 |
| 레인저 🏹 | AGI·END 5, 야생 4회 | 채집 효율 상승 |
| 드루이드 🌿 | FAI 4·END 6, 야생 5회 | 약초·농산물 생산 |
| 현자 📚 | INT 7, 마법 6회 | 유물 감정 |
| 상인 💰 | CHA 6, 교역 6회 | 시장 가격 영향력 |
| 팔라딘 ⚜ | STR·FAI 5, 선 성향 | 헌금·기부 이벤트 |
| 네크로맨서 💀 | INT 6, 악 성향 | 암시장 재료 수요 |

### 💞 관계 시스템
16가지 관계 유형 (친구, 전우, 라이벌, 연인, 배우자, 채권자/채무자, 맹약 등)  
친밀도가 누적되면 **연애 → 결혼**으로 자연 발전하며, 이야기 흐름 속도 슬라이더로 진행 속도를 조절할 수 있습니다.

관계도는 **목록 보기** 또는 **SVG 네트워크 그래프**로 확인 가능하며, 사망한 캐릭터도 💀로 표시됩니다.

### 📊 시장 경제
- 수요·공급에 따른 **동적 가격 변동**
- 직업별로 시장에 재료를 공급하거나 소비
- 세계 위협도가 높아질수록 **물가 상승** (최대 2배)
- 마을 창고(`townGold`)에 금화가 쌓여 건물 건설 재원으로 활용

### 🏰 거점 & 건물
거점은 **야영지 → 마을 → 성채 → 왕도** 4단계로 성장합니다.  
건설 가능한 건물 10종:

| 건물 | 효과 |
|------|------|
| 🏗 공동 창고 | 자재 채집량 +50% |
| ⚔ 모험가 길드 | 의뢰 수입 증가, 전직 조건 완화 |
| 🏪 상점 | 시장 물가 5% 할인 |
| 🏨 여관 | HP 회복, 피로 감소 |
| ⛪ 성당 | 이성 회복, 상태이상 해제 |
| 🥊 훈련장 | STR·END 성장 가속 |
| ⚒ 대장간 | 철광석 채집량 +50% |
| 📚 도서관 | INT 성장, 마법 경험치 증가 |
| 🗼 감시탑 | 세계 위협도 자연 감소 |
| 🎪 광장 | 사교 이벤트 빈도 증가 |

캐릭터들이 **확률적으로 건물을 방문**하여 효과를 받고 마을 금고에 금화를 지출합니다.

### 🌍 세계 위협도
0~100 범위의 위협도가 매일 변화하며 5단계로 세계 상황이 달라집니다.

| 단계 | 이름 | 시장 배율 |
|------|------|----------|
| 0~20 | 평화로운 왕국 | ×1.0 |
| 21~40 | 불안한 징조 | ×1.1 |
| 41~60 | 전쟁의 기운 | ×1.25 |
| 61~80 | 어둠의 확산 | ×1.5 |
| 81~100 | 마왕의 강림 | ×2.0 |

### 🎭 상태이상
| 이상 | 효과 |
|------|------|
| 💀 저주 | 전 스탯 -2 |
| 😱 공포 | STR·AGI -3, 전투 기피 |
| ☠ 중독 | HP -5/일 |
| 😵 혼란 | CHA 무효화 |
| 🌀 광기 | CHA 무효 + 행동 왜곡 |
| 💕 홀림 | 무의식적 금화 지출 |

### 🏁 9가지 엔딩
세계 위협도, 생존 캐릭터 수, 성장 지표에 따라 다양한 엔딩이 발생합니다.

## 조작법

| 버튼 | 기능 |
|------|------|
| ▶ Next Day | 하루 진행 |
| ⚡ Next Event | 다음 중요 이벤트(사망·전직·스토리 선택)까지 자동 진행 |
| 📖 속도 슬라이더 | 이야기 흐름 속도 조절 (0.5× ~ 3×) |
| ＋ 캐릭터 추가 | 새 캐릭터 생성 |
| 💾 저장 / 📂 불러오기 | 로컬스토리지 저장 |

## 기술 스택

- **Vanilla JS** — 프레임워크 없음, 빌드 도구 없음
- **CSS Custom Properties** — 다크/라이트 테마
- **LocalStorage** — 게임 세이브
- **SVG** — 관계도 네트워크 그래프
- **Surge.sh** — 정적 호스팅

## 로컬 실행

```bash
# 웹서버 없이 바로 열기 (크롬 --allow-file-access-from-files 필요)
# 또는 간단한 서버 실행:
npx serve .
```

파일 구조:
```
FWS/
├── index.html
├── css/style.css
└── js/
    ├── config.js   # 상수·데이터 정의
    ├── game.js     # 핵심 게임 로직
    ├── events.js   # 이벤트 테이블
    ├── market.js   # 시장 경제
    ├── ui.js       # 렌더링
    ├── storage.js  # 세이브/로드
    └── main.js     # 진입점·이벤트 바인딩
```
