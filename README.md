# ⚔ Fantasy World Simulator

> 캐릭터들이 살아 숨쉬는 판타지 세계를 운영하는 텍스트 기반 시뮬레이터

**[▶ 플레이하기](https://fws-sim.surge.sh)**

---

## 개요

Fantasy World Simulator는 플레이어가 캐릭터를 설계하고, 그들이 스스로 모험을 떠나며 성장하는 과정을 지켜보는 **방치형 시뮬레이션 게임**입니다.  
캐릭터들은 매일 서로 관계를 맺고, 직업을 얻고, 파티를 꾸리고, 연애를 하고, 마왕군의 침공을 막아내며 세계의 운명을 결정짓습니다.  
목표는 **최대한 오래 버티는 것**. 거점 내구도가 0이 되거나 전원이 전사하면 게임이 끝납니다.

---

## 주요 시스템

### 🧑 캐릭터

- **6가지 스탯** — STR(전투) · INT(마법) · FAI(신성) · AGI(민첩) · CHA(사교) · END(생존)
- **성별 / MBTI / 성향(선·중립·악) / 초기 정신 상태** 설정
- **이모지 아이콘** — 성별별 풀에서 선택
- 스탯은 이벤트·훈련·아이템 패시브로 성장하며, 레벨업 시 스킬이 한 단계 상승합니다

### ⚔ 12가지 직업 (전직 시스템)

스탯과 행동 횟수 조건을 충족하면 전직 제안 팝업이 발생합니다. 전직 비용은 150G(초전직) / 300G(재전직).

| 직업 | 주력 스탯 | 특성 |
|------|----------|------|
| 전사 | STR | 높은 HP·공격력, 전선 담당 |
| 마법사 | INT | 광역 스킬, 마법 소재 활용 |
| 성직자 | FAI | 치유·해제 스킬, 포션 제조 |
| 도적 | AGI | 은신·암시장 접근 |
| 기사 | STR·END | 균형형, 안정적 전투력 |
| 음유시인 | CHA | 팀 사기 상승, 사교 이벤트 강화 |
| 레인저 | AGI·END | 원거리, 야생 생존 |
| 드루이드 | END | 약초 포션 제조, 자연 버프 |
| 현자 | INT | 강력 마법, 마법서 해독 |
| 상인 | CHA | 시장 거래·협상, 자금 관리 |
| 팔라딘 | FAI·STR | 신성 공격 + 자기 치유 |
| 네크로맨서 | INT | 강력하지만 위협도 상승 부작용 |

### 💞 관계 시스템

- **16가지 관계 유형** — 친구, 전우, 라이벌, 연인, 배우자, 맹약, 채권자/채무자, 고용주/피고용인 등
- MBTI 궁합과 호감도에 따라 **친구 → 연인 → 결혼** 자연 발전
- 결혼한 두 캐릭터는 **맹약**을 맺어 위기 시 HP를 분담
- **바람·배신 이벤트** 발생 가능 (순애 모드 ON 시 차단)
- 설정에서 동성/이성 커플 허용 여부 별도 설정 가능
- **관계도 그래프** — 전체 인물관계를 SVG 네트워크로 시각화 (직업 약자 + 관계선 색상 구분)

### 🤝 파티 시스템

- 호감도 45 이상인 두 캐릭터가 자동으로 파티를 결성
- 파티당 최대 4명, 결성 직후 **퀘스트 선택지** 제시
- **12~17일 주기**로 새 퀘스트 제안 (combat / explore / defend / rest 타입)
- 등급 D~S, 성공률·보상·위협도 감소가 모두 다름
- 퀘스트 보상은 **연도별 스케일링** (Y1: ×1.0 → Y5+: ×3.0)
- 파티원 피로 과부하, 내부 갈등 등으로 자연 해산 가능

### 🏰 거점 & 건물

거점은 **야영지 → 마을 → 성채 → 왕도** 4단계로 업그레이드됩니다.  
건물 건설 비용은 목재·철광석·마법 결정·금화로 지불하며, 캐릭터들이 확률적으로 방문해 효과를 받습니다.

| 건물 | 효과 | 해금 조건 |
|------|------|----------|
| 🏗 공동 창고 | 자재 자동 채집 | Lv1 |
| ⚔ 모험가 길드 | 의뢰 수입, 길드 퀘스트 활성화 | Lv2 |
| 🏪 상점 | 소모품·식량 자동 구매, Lv업 시 신규 아이템 해금 | Lv2 |
| 🏨 여관 | HP +15, 피로 -20 | Lv2 |
| 🥊 훈련장 | STR·END 수련 누적 | Lv2 |
| ⛪ 성당 | 이성 +5, 상태이상 해제 | Lv3 |
| ⚒ 대장간 | 철광석 채집, 장비 자동 제작, Lv업 시 강화 성공률 상승 | Lv3 |
| 📚 도서관 | INT 수련, EXP +2 | Lv3 |
| 🗼 감시탑 | 위협도 매일 자동 감소 (패시브) | Lv3 |
| 🎪 광장 | 동료 호감도 +1 | Lv4 |

> **상점·대장간만** 업그레이드 가능 (Lv1~5). 자원이 충분하면 거점 탭이 빛납니다.

### ⚒ 장비 강화 시스템

대장간 건설 시 장착 장비를 자동으로 강화 시도합니다 (매일 12% 확률).

| 강화 구간 | 강화당 스탯 상승 |
|----------|---------------|
| +1 ~ +3 | +1 |
| +4 ~ +5 | +2 |
| +6 ~ +8 | +3 |
| +9 ~ +10 | +5 |

강화 실패 시 골드만 소모, 고강화 구간에서는 장비 파괴 위험이 있습니다.  
장비 보너스는 **전투 공격력·maxHP 계산에 실시간 반영**됩니다.

### 🧪 제작 시스템

- **드루이드·성직자·팔라딘** — 약초 3개 → 치유 포션 제조 (15% 확률 강화 포션)
- **마법사·현자·네크로맨서** — 마석 4개 → 마법 두루마리 제조 (20% 확률 화염 스크롤)

### 📊 시장 경제

- `rawPrice = basePrice × (demandIndex / supplyIndex) × regionFactor × stabilityFactor`
- 가격 범위: 기본가 **×0.3 ~ ×5** 고정 클램프
- 위협도 상승 → 무기·방어구 물가 상승, 식량 물가 소폭 상승
- **인플레이션**: 캐릭터 평균 골드 2000G 초과 시 소모품 수요지수 +2%/일
- **디플레이션**: 평균 골드 1200G 미만 시 수요지수 100으로 -3%/일 회귀
- **거점 레벨** 높을수록 `stabilityFactor`로 물가 안정
- 판매 시 거점 레벨에 따라 **5~8% 시장 수수료**가 공동 금고로 적립

### 🌍 세계 위협도

| 단계 | 이름 | 시장 배율 |
|------|------|----------|
| 0~20 | 평화로운 왕국 | ×1.0 |
| 21~40 | 불안한 징조 | ×1.1 |
| 41~60 | 전쟁의 기운 | ×1.25 |
| 61~80 | 어둠의 확산 | ×1.5 |
| 81~100 | 마왕의 강림 | ×2.0 |

- **매일 자연 증가** (1년차 +0.05/일 → 8년차+ +1.10/일)
- **70 이상**: 위기 선택 이벤트 발동 (30일 쿨다운)
- **계절 말 (90일마다)**: 마왕군 계절 침공 — 위협도에 따른 전투, 패배 시 거점 내구도 -1

### ⚔ 계절 침공

90일마다 마왕군이 침공합니다. 생존 캐릭터 전원이 응전하며, 스킬을 **전략적으로 선택**합니다.
- 아군 평균 HP 50% 미만 → 힐 스킬 우선 사용
- 1라운드 → 버프 스킬 우선
- 그 외 → 가장 높은 atkBonus 스킬 선택

## 기술 스택

- **Vanilla JS** — 프레임워크 없음, 빌드 도구 없음
- **CSS Custom Properties** — 다크/라이트 테마
- **LocalStorage** — 게임 세이브
- **SVG** — 관계도 네트워크 그래프
- **Surge.sh** — 정적 호스팅

## 로컬 실행

```bash
npx serve -p 5174 .
# 또는
npx surge . fws-sim.surge.sh   # 배포
```

파일 구조:
```
FWS/
├── index.html
├── css/style.css
└── js/
    ├── config.js    # 상수·데이터 정의 (클래스·아이템·건물·MBTI 등)
    ├── game.js      # 핵심 게임 로직 (이벤트 처리·전투·경제·관계)
    ├── events.js    # 개인 이벤트 테이블 + 상호작용 이벤트
    ├── market.js    # 시장 경제 (가격 계산·인플레이션·이상 감지)
    ├── ui.js        # 렌더링 (캐릭터 카드·관계도·거점·설정)
    ├── storage.js   # 세이브·불러오기
    └── main.js      # 진입점·이벤트 바인딩
```
